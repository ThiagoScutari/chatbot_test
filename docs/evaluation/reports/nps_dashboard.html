O que é o NPS?

O Net Promoter Score mede a lealdade dos clientes com uma única pergunta: "De 0 a 10, o quanto você recomendaria nossa empresa?" Promotores (9-10) são clientes entusiastas. Detratores (0-6) podem prejudicar a reputação. Neutros (7-8) não influenciam o resultado.
